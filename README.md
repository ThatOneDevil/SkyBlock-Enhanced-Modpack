# Skyblock Enhanced: Modern Edition

Skyblock Enhanced: Modern Edition is an all-in-one modpack for playing Hypixel Skyblock on the latest Minecraft version, 1.21+.  
Building on the success of [Skyblock Enhanced [Hypixel]](https://modrinth.com/project/9JTbeXjU) (over 35,000 downloads), this pack brings the best set of mods to play Hypixel Skyblock with better performance and smoother gameplay.

## Key Features

- Designed for Hypixel Skyblock in Minecraft 1.21+  
- Skyblocker mod for essential Skyblock features  
- Sodium and other performance mods for higher FPS  
- Easier to run than 1.8.9 while keeping most of the same Skyblock-specific features  
- Ready for the upcoming Hypixel Forging Update  
- More features to come as mods update
