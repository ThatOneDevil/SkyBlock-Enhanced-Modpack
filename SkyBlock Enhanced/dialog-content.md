## üö® Important Notes

- Extracting a config will **overwrite your mod settings**, unless this is your first time starting the pack.
- Make a backup using `/packcore archive` before extracting if you want to keep your current setup.
- You can re-enable this dialog anytime with `/packcore dialog true`, or in ModMenu under  
  **Show Config Selection Dialog On Start Up**.  
  This is useful if you want to switch or reset your config later.
